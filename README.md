# B581-Tweets

B581-Tweets is a simple Twitter-style blogging app built with **Vue 3**, **Pinia**, **Bootstrap**, and **Node.js / Express** backend with **MongoDB**. Users can create posts, view posts, comment, and admins have additional control over the platform.  

---

## Features

### General Features
- User registration and login.
- Create, read, edit, and delete blog posts (CRUD).
- Add and delete comments on posts.
- Responsive design for desktop and mobile.
- Twitter-like layout and styling.

### Admin Features
- Access the **Admin Dashboard**.
- View statistics (total posts, total users).
- Delete any post or comment.

### Authentication
- Protected routes for users and admins.
- JWT-based authentication stored in Pinia and `localStorage`.

## Tech Stack

- **Frontend:** Vue 3, Pinia, Vue Router, Bootstrap 5
- **Backend:** Node.js, Express
- **Database:** MongoDB (Atlas)
- **Authentication:** JWT (JSON Web Token)
- **Styling:** Twitter-inspired UI with Bootstrap
